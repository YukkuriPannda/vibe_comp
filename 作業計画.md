# ベース用コンプレッサー 開発計画

作成日: 2026-07-01
目的: ベースギター用コンプレッサーペダルの回路設計 → 試作検証 → KiCad基板設計 → JLCPCBへの基板製造依頼

## 前提（今回決定した方針）

- 回路方式: **OTA方式**（Ross Compressor / Dyna Compタイプ。LM13700ベース）
  - 理由: 部品点数が少なく安価、実績豊富、部品調達がしやすい（CA3080は生産終息傾向のため、入手性の良い**LM13700**（デュアルOTA）を採用）
- JLCPCB依頼範囲: **基板製造のみ（Fab）**。実装（PCBA/SMT組立）は依頼せず、部品は自分でLCSC/秋月/Mouser等から調達し手ハンダする
  - → 部品はスルーホール（THT）中心で設計し、手ハンダしやすくする

## 確定仕様（2026-07-01 決定）

- **電源**: 9V電池駆動のみ（DCジャックなし。電池スナップ直結、必要ならバッテリークリップ用の基板パターンを用意）
- **LED**: あり（バイパス状態インジケーター、後述のラッチ回路の状態を表示）
- **対象音域**: 4弦ベース前提（低音E ≈ 41Hz）。入力ハイパスのカットオフはこれを減衰させない値（目安 10Hz台〜20Hz程度）に設定し、エンベロープ検出側の時定数も4弦の基音を追従できるレンジで調整する
- **筐体**: 1590B。ただし通常の「パネルにポット/スイッチを取り付けて配線」ではなく、**基板を露出させるデザイン**（1590Bに基板がそのまま収まる/窓から見える構成、パネル配線レス）
- **ノブ構成**: 最小限の **Sustain（圧縮量）+ Level（出力）の2ノブ**。Attack/Releaseは回路内で固定値（Phase 2/3で追い込む）
- **ポット**: 基板実装型ロータリーポット（Alpha RD901F等、指でつまみを回せるPCBマウントタイプ）を採用し、配線レスで基板に直付け。露出デザインと相性の良い部品を選定
- **オン/オフスイッチ（バイパス）**: 通常のフットスイッチではなく **Cherry MX準拠のキースイッチ**を使用
  - Cherry MXはモーメンタリ（押している間だけON）のため、そのままではトゥルーバイパスのラッチ動作ができない
  - → **タクトスイッチ的な入力として使い、CMOSフリップフロップ（4013等）でラッチしてリレーを駆動する「ソフトタッチ・トゥルーバイパス」回路**を追加する（信号経路自体はリレーが機械的に直接切り替えるためトゥルーバイパス）
  - LEDはこのフリップフロップの状態を表示するドライブ回路と共用
  - Cherry MX用のPCBフットプリント（5pin/plate mount想定）をKiCadで用意する

## 作業フェーズ

### Phase 0: 仕様確定 ✅ 完了（2026-07-01）
- ノブ構成・ポット種別・バイパススイッチ方式・電源・LED・筐体・対象音域を確定（上記「確定仕様」参照）
- 参考回路（Ross Compressor の解析記事: Electrosmash、General Guitar Gadgetsの配布回路図等）を突き合わせてベース向けに調整する定数（入力カップリングコンデンサ容量、エンベロープの時定数）の方針を決める

### Phase 1: 回路設計（部品レベルの仕様確定）
- **コンプレッサー本体**: OTA(LM13700) + デュアルオペアンプ(TL072等、ノイズ・入手性で選定) によるゲインリダクション段、整流（エンベロープ検出）段、入出力バッファの回路構成を確定。ノブはSustain/Levelの2つのみに絞った構成にする
- **バイパス部**: Cherry MXキースイッチ（モーメンタリ入力）→ デバウンス → CMOSフリップフロップ（4013等）→ 小信号リレー駆動 によるソフトタッチ・トゥルーバイパス回路、およびLEDドライブを設計
- 電源は9V電池単電源、仮想グラウンド（半電源バイアス）方式。電池残量を考慮した低消費電流設計（リレー・LED電流に注意）
- 部品はJLCPCB提携部品庫（LCSC）での在庫・入手性を確認しながら選定。ポットは基板実装型ロータリーポット（Alpha RD901F等）、スイッチはCherry MX用フットプリントを使用

### Phase 2: シミュレーション検証（LTspice / ngspice）
- LM13700のSPICEモデルを用いてゲインリダクション動作、アタック/リリース特性をシミュレーションで確認
- 実機を組む前にここで論理的な誤りを潰す（コンプレッサーは帰還の極性を間違えるとエキスパンダー的挙動になるため要注意）

### Phase 3: ブレッドボード/ユニバーサル基板での試作
- 実際にベースを繋いで圧縮感・ノイズ・発振の有無を確認
- 定数（アタック/リリースの時定数、コーナー周波数）を実測しながらベース向けに追い込む
- **この段階で「ちゃんと圧縮する」ことを確認してから基板発注に進む**（基板を先に発注して動かないと手戻りが大きいため）

### Phase 4: KiCad回路図・基板レイアウト設計
- 試作で確定した回路をKiCad 8で回路図化（インストール確認済み: KiCad 8.0.1, kicad-cli あり）
- ERC（電気ルールチェック）を実行しエラーゼロを確認
- **基板露出デザイン**のため、1590Bの内寸に収まる外形サイズで設計し、ポット（Alpha RD901F）とCherry MXキースイッチの軸/キーキャップが1590Bの窓・開口部から出る位置にレイアウト
- 基板外形はねじ止め用の取付穴（1590Bの固定方法に合わせる）も考慮
- PCBレイアウト、DRC（デザインルールチェック）実行

### Phase 5: DFM（製造性）チェック
- JLCPCBの基板製造ルール（最小配線幅/間隔、最小ドリル径、外形寸法公差など）に適合しているか確認
- 2層基板・板厚1.6mm・HASL仕上げ等、標準スペックで収まるようにする（コスト最小化）

### Phase 6: ガーバー出力・JLCPCB発注
- KiCadから「JLCPCB用ガーバー出力」プラグイン or 手動プロットでガーバー/ドリルファイルを生成しZIP化
- JLCPCBサイトにアップロードし、以下を指定して発注
  - 枚数（試作は5枚が最小ロット）
  - 層数2層、板厚1.6mm、レジスト色、シルク色、表面処理（HASL推奨・安価）
  - 実装は依頼しない（PCBAオプションはOFF）のままチェックアウト
- 送料・納期を確認して発注確定（実際の支払い・発注操作はユーザー本人が実施）

### Phase 7: 部品調達（基板以外） ✅ 秋月での購入可否チェック完了（2026-07-01, 詳細は `bom/部品リスト.md`）
- **秋月電子で購入**: TL072相当（`TL072G-D08`/`NJM072D`）、4013相当（`TC4013BP`）、1N4148、単連10kΩ基板用ロータリーポット（`RK09D117000B/C`, ※Alpha RD901Fの直接代替品）、5VリレーDPDT（`941H-2C-5D`）、スイッチ付き6.3mmジャック（`MJ-188LPD-8C`, 電池ON/OFF兼用）、1590B代替ケース（`TD6-11-3N`, 115.5×64.5×31.8mm）、抵抗/コンデンサ/LED/ICソケット/電池スナップ等一般部品
- **秋月では買えない分はAmazon.co.jpでまとめて調達（送料を2箇所に集約、Mouser発注は基本不要）**: LM13700N、Alpha RD901F（必要な抵抗値そのままで購入可）、1590B（実物 or 同等サイズ）、2N5457（必要な場合）。ただしLM13700はマーケットプレイス品の品質リスクがあるため、信頼性優先なら1点だけMouser日本の正規品`LM13700N/NOPB`にする選択肢も残す
- **キースイッチ（Cherry MX互換）**: ユーザー手持ち品を使用のため調達不要
- 2N5457相当のJFETが必要になった場合は秋月の後継品`2SK208`シリーズで代替（Phase1で必要性を精査）

### Phase 8: 組み立て・動作確認・調整
- 手ハンダで実装、動作確認、必要ならトリマーで微調整

# 回路設計（Phase 1 確定版 / 2026-07-01）

**重要**: この回路はデータシートの式・KiCad標準シンボルのピン配置と照合して設計したオリジナル設計であり、既存の特定ペダル(Ross Compressor等)のコピーではない。**まだSPICEシミュレーション(Phase2)・ブレッドボード試作(Phase3)による実機検証を行っていない。** 基板発注(Phase6)の前に必ずPhase2/3を実施すること。特にコンプレッサーは帰還の極性を間違えると圧縮ではなく発振・エキスパンダー的動作になるため、試作確認は必須。

## 使用ICのピン配置（データシート/KiCad標準シンボルで確認済み）

### LM13700（16pin DIP, デュアルOTA）— TIデータシート(SNOSBW2F)の "Pin Functions" 表で確認
| Pin | 機能 | Pin | 機能 |
|---|---|---|---|
| 1 | IABC1（ゲイン制御電流入力） | 9 | Buffer Output 2 |
| 2 | Diode Bias 1 | 10 | Buffer Input 2 |
| 3 | Non-inverting Input 1 (+) | 11 | V+ |
| 4 | Inverting Input 1 (−) | 12 | Output 2 |
| 5 | Output 1 | 13 | Inverting Input 2 (−) |
| 6 | V− | 14 | Non-inverting Input 2 (+) |
| 7 | Buffer Input 1 | 15 | Diode Bias 2 |
| 8 | Buffer Output 1 | 16 | IABC2 |

今回は**OTA#1（pin1-8）のみ使用**、OTA#2は未使用（後述の通り無効化）。

### TL072（8pin DIP, デュアルオペアンプ）— 標準ピン配置
1=Out1, 2=In1(−), 3=In1(+), 4=V−, 5=In2(+), 6=In2(−), 7=Out2, 8=V+

### TC4013BP（14pin DIP, デュアルDフリップフロップ）— 標準ピン配置
1=Q1, 2=/Q1, 3=CLK1, 4=RESET1, 5=D1, 6=SET1, 7=VSS(GND), 8=SET2, 9=D2, 10=RESET2, 11=CLK2, 12=/Q2, 13=Q2, 14=VDD

今回は**フリップフロップ#1のみ使用**（トグル動作: /Q1をD1に帰還）。

## 全体ブロック構成

```
入力Jack(J1,電池ON/OFF兼用)
  → 入力バッファ(U1A)
  → OTAゲイン制御段(IC1 LM13700 OTA#1、Sustainで帰還量を調整)
  → メイクアップゲイン段(U1B)
  → Levelポット(VR2)
  → 出力Jack(J2)

OTA出力ノード → エンベロープ検出(D1,C_hold等) → Sustainポット(VR1) → Q_ctrl(NPN) → IABC1(帰還でゲイン低下=圧縮)

Cherry MXスイッチ(SW1、手持ち部品) → デバウンス → TC4013 Dフリップフロップ(トグル)
  → Q_relay(NPN)駆動 → リレー(RL1, DPDT) → 信号経路をトゥルーバイパス切替
  → LED1(フリップフロップの状態を表示)
```

## 4. エンベロープ検出（Sustain制御ループ）

OTA出力ノード(pin5/RLの接続点)からタップし、圧縮の判定に使う信号を作る。

| 部品 | 値 | 接続 |
|---|---|---|
| C_env | 0.1µF | pin5ノード → ノードA |
| R_envbias | 1MΩ | ノードA → GND（ノードAのDC基準を0Vに設定。AC信号のみ通す） |
| D1 | 1N4148 | アノード=ノードA、カソード→R_attack |
| R_attack | 2.2kΩ | D1カソード → C_hold（アタックタイム τ≈2.2ms） |
| C_hold | 1µF | R_attack後段 → GND（エンベロープ保持コンデンサ） |
| R_release | 220kΩ | C_holdと並列（リリースタイム τ≈220ms） |
| VR1 (Sustainポット) | 10kΩ（秋月 `RK09D117000B` 等） | C_hold後段 → ワイパー → R_base(2.2kΩ) → Q_ctrlベース。**電圧分圧のスケーラーとして使用**（直列可変抵抗ではなく、検出電圧がQ_ctrlベースに伝わる比率を調整する構成。秋月で入手できる10kΩ単連ポットでも十分機能するよう設計） |

**Attack/Release値は固定**（前回の仕様確定どおり、ノブはSustain/Levelのみ）。Phase2/3で実際に鳴らしながら微調整可能なようR_attack/R_releaseは基板上でランド互換の値違い品に差し替え可能な配置にしておくと良い。

| 部品 | 値 | 接続 |
|---|---|---|
| C3 | 1µF | pin5(OTA出力)ノード → U1B pin5(+) |
| R5 | 100kΩ | U1B pin5(+) → Vref |
| R6 | 10kΩ | U1B pin6(−) → GND |

## 6. 出力段

| 部品 | 値 | 備考 |
|---|---|---|
| VR2 (Levelポット) | 10kΩ Aカーブ（秋月`RK09D117000C`） | C4後段-GND間、ワイパーが出力 |
| C5 | 1µF | VR2ワイパー → J2(出力Jack)、DCブロック |
| J2 | 6.3mm標準ジャック（スイッチなし） | 出力専用 |

出力バッファは設けない（後段の入力インピーダンスが十分高いことを前提とした一般的なパッシブ出力段。TL072は2回路とも上記で使い切っているため追加不可）。
